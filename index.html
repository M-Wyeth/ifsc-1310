Progressive Enhancement

Progressive Enhancement is the technique of building web pages with a focus on content first. By focusing on content and ensuring that, regardless of browser or device, the content is available to all users, you create for stability. By only using the bare minimum of externally linked CSS and JavaScript, the designer ensures that the content will be accesssable by anyone trying to access the web page. Progressive enhancement is important because it does not matter how fancy you make it look with CSS and JavaScript if is someone is unable to access the content. By ensuring the base content is easily accessible through a solid HTML structure unaffected by the CSS or JS, you are able to reach as many people as possible without fear of everything breaking.
